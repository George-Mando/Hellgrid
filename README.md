## 🌐 `DEPLOY YOUR OWN`

Want to play **Hellgrid** or **Birdy Blocks** directly from this repo?

```bash
# Clone the repo
git clone https://github.com/George-Mando/Hellgrid.git
cd Hellgrid

# Open directly in browser — no build step needed!
open featherfall-village.html   # macOS
start featherfall-village.html  # Windows
xdg-open featherfall-village.html # Linux
```

> Both games are **single HTML files** — no dependencies, no install, works offline. Just open and play.
